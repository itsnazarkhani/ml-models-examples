# الگوریتم‌های یادگیری ماشین
در این مخزن، من مجموعه‌ای از پروژه‌های یادگیری ماشین را قرار می‌دهم. هر پروژه یک الگوریتم خاصی از یادگیری ماشین با استفاده از کتابخانه‌هایی همچون numpy, pandas, matplotlib و scikit-learn پیاده سازی می‌کند. علاقه‌مندان به یادگیری ماشین می‌توانند با استفاده از این پروژه‌ها نحوه پیاده سازی این الگوریتم‌ها را مشاهده نمایند.

از هرگونه مشارکت شما عزیزان در رفع ایرادات یا افزودن الگوریتم‌های جدید، کمال تشکر و قردانی را دارم.

## پروژه‌های موجود

1. [**دسته‌بند درخت تصمیم**](/decision_tree/decision_tree.ipynb)
   - پیاده‌سازی پایه‌ای از دسته‌بند درخت تصمیم از ابتدا.
   - دیتاست: دیتاست گل‌های Iris.
   - مفاهیم پوشش داده شده: ساخت درخت، شاخص جینی، آنتروپی، کسب اطلاعات.

## هدف

این مخزن به‌عنوان یک محیط آزمایشی برای بررسی تکنیک‌های یادگیری ماشین و درک نحوه عملکرد آن‌ها در سطح بنیادی طراحی شده است. هر پروژه به گونه‌ای طراحی شده که:
- ساده و قابل فهم باشد.
- بر روی مفاهیم خاص یادگیری ماشین متمرکز باشد.
- مناسب برای مبتدیان باشد.

## نحوه استفاده

مخزن را کلون کنید و پروژه‌ها را روی سیستم خود اجرا کنید:

```bash
# کلون کردن مخزن
git clone https://github.com/your-username/ml-models-examples.git

# نصب Jupyter Notebook
pip install notebook

# رفتن به پوشه پروژه و اجرای کد، برای مثال:
cd ml-models-examples/decision-tree
jupyter decision_tree.py

# درصورت لزوم کتابخانه‌های مورد نیاز پروژه نصب شود.
```
